# Rancho Lechero - El Valle Lechero

Página web corporativa para el Rancho Lechero "El Valle Lechero", ubicado en Los Altos de Jalisco, México.

## Características

- **Diseño Responsivo**: Optimizado para desktop, tablet y móvil
- **Navegación Moderna**: Menú fijo con navegación móvil
- **Secciones Completas**:
  - Hero con imagen de fondo
  - Sobre nosotros con estadísticas
  - Instalaciones y equipos
  - Catálogo de ganado
  - Productos con filtros
  - Galería con lightbox
  - Noticias del rancho
  - Formulario de contacto
  - Footer completo

## Tecnologías

- HTML5 semántico
- CSS3 con variables y Grid/Flexbox
- JavaScript vanilla para interactividad
- Fuentes de Google Fonts
- Imágenes de Unsplash

## Estructura de Archivos

```
/
├── index.html          # Página principal
├── style.css           # Estilos CSS
├── script.js           # JavaScript funcional
├── data.js            # Datos de productos, galería y noticias
└── README.md          # Este archivo
```

## Cómo Usar

1. Abre `index.html` en tu navegador web
2. Navega por las diferentes secciones usando el menú
3. En la sección de productos, usa los filtros para ver categorías específicas
4. Haz clic en las imágenes de la galería para verlas en tamaño completo
5. Completa el formulario de contacto

## Personalización

- **Colores**: Modifica las variables CSS en `:root` en `style.css`
- **Contenido**: Edita los textos en `index.html` y los datos en `data.js`
- **Imágenes**: Reemplaza las URLs de Unsplash con tus propias imágenes
